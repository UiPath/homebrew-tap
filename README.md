# UiPath Homebrew Tap

Homebrew tap for UiPath CLI tools.

## Install

```bash
brew tap uipath/tools
brew install uip
```

Or in a single command:

```bash
brew install uipath/tools/uip
```

## Upgrade

```bash
brew upgrade uip
```

## Uninstall

```bash
brew uninstall uip
```

## Available Formulas

| Formula | Description |
|---------|-------------|
| `uip` | UiPath CLI for automation lifecycle management |
